## Problem 2

Create a function called *CrazySum(s,n)* which takes in two numbers $$s$$ and $$n$$ and returns

$$\frac{1}{s-1} + \frac{1}{s-2} + \frac{1}{s-3} + \frac{1}{s-4} + \dots + \frac{1}{s-n}.$$

Be sure to carefully document your function.
