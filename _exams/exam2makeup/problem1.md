## Problem 1

Create a function called *goodsign(x)* which takes in a number $$x$$

and returns $$1$$ if either $$\sin(x)\geq 1/2$$ or $$\sin(x)\leq -1/2$$, and returns $$0$$ otherwise.

**Be sure to carefully document your function.**
