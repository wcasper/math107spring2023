## Problem 3

Create a function called *powerMat(n)* which takes in a positive integer $$n$$ and returns the $$n\times n$$ matrix $$A$$ whose $$(j,k)$$'th entry is

$$A(j,k) = 2^{j-k}.$$

Be sure to carefully document your function.
