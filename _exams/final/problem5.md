## Problem 5


Write a MATLAB function called *finalMAT(m,n)* that uses conditionals and nested for loops to create an $$m\times n$$ matrix $$A$$ whose $$(j,k)$$ entry is

$$A(j,k) = \left\lbrace\begin{array}{cc}
jk &  j>k\\
1 & j = k\\
0 & j < k
\end{array}\right.$$
