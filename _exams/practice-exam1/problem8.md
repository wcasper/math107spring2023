## Problem 8

Consider the following code

```
error = 1.0;
x = 1
while(error > 0.01)
  x = x - x/2 + 1/x;
  error = abs(x*x-2);
end
```
Determine the values of *x* and *error* at the end of the code block.  Be sure to carefully show your work.
